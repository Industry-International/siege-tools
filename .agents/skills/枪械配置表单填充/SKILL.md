# 枪械配置表单填充

根据用户输入的枪械名称和职业对应关系，自动查询数据并填充职业配置表。

## 1. 前置资源

- **枪械/配件数据目录**: `./tacz所有的枪械配件数据/`
- **职业配置表目录**: `server_scripts/profession/prof_configs/`
- **职业使用教程**: `使用指南/profession-使用教程.md`
- **特殊武器查询**: `给AI阅读的文档/.probe/` 或翻阅模组文件
- **TaCZ 枪包数据**: `./tacz所有的枪械配件数据/`

## 2. 工作流程

### 2.1 查询枪械
- 根据输入的枪械名称，在 `tacz所有的枪械配件数据` 下查询是否存在该枪械
- 如果**没有**找到同名枪械，则判断为不存在，**不进行任何操作**

### 2.2 检查配置表
- 检查 `server_scripts/profession/prof_configs/` 下对应的配置表
- 如果表中**已有**该枪械的配置，则**不进行任何操作**（跳过已配置的）

### 2.3 填写表单
- 根据 `使用指南/profession-使用教程.md` 的指引填写配置
- 根据用户提供的枪械与职业对应关系进行配置

### 2.4 配件配置规则
- **配件给到最满状态**（用户会后续自行删减）
- 判断逻辑：
  - 根据配件支持的枪械类型
  - 根据配件安装位置
  - 根据枪械支持的槽位
  - 三者都匹配才安装

### 2.5 不可使用配件
以下配件不可使用：
- `lavender:scope_rifles_x6`

### 2.6 特殊武器处理
- 如果用户输入的是职业对应的**特殊武器**
- 需要去写**特殊武器表**（不是普通配置表）
- 特殊武器一般是**卓越前线**的武器
- 查询途径：`给AI阅读的文档/.probe/` 或直接翻阅模组

## 3. 重要原则

- 输入中没有同名枪械 → 跳过
- 配置表中已有该枪械 → 跳过
- 配件尽可能给满 → 用户自行删减
- 特殊武器走特殊武器表
