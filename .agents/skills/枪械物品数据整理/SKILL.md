# 枪械物品数据整理

将所有 TaCZ 枪械和配件数据整理成结构化的表格，存放于 `./tacz所有的枪械配件数据/` 目录下。

## 1. 输出目标

所有表格输出到 **`./tacz所有的枪械配件数据/`** 目录下。

### 1.1 枪械表
每条数据应包含：
- 枪械图片
- 枪械名
- 对应的 NBT 标签（GunId）
- 枪械弹药类型（如 `tacz:762x39`）
- 枪械支持的配件槽位

### 1.2 配件表
每条数据应包含：
- 配件图片
- 配件名
- 对应的 NBT 标签（AttachmentId）
- 配件支持的枪械类型

### 1.3 组织方式
- 枪械和配件可以拆分为不同的表
- 也可以按照不同枪包来拆分（按枪包分文件）

## 2. 工作流程

### 2.1 检查已有数据
- 如果 `./tacz所有的枪械配件数据/` 下已有成型的表
- 则**仅做增删修改**，不要重构整个表

### 2.2 脚本方式获取
- 如果使用脚本来获取数据
- 脚本可以保留在 `./tacz所有的枪械配件数据/` 目录下
- 每次执行前，如果检查到已有预留脚本，**先查看脚本内容**

### 2.3 需要查询的数据来源
- 通过 ProbeJS 的 `.d.ts` 文件（`给AI阅读的文档/.probe/`）
- 翻阅模组文件/枪包文件（`../tacz/`）
- 游戏内实际测试验证

## 3. 数据覆盖范围

- 所有枪包中的枪械
- 所有枪包中的配件
- 配件与枪械的兼容关系
