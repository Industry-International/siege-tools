# 晴雪UI 开发指引

## 0. ⚠ 优先级说明

**LDLib2 是本项目编写自定义 GUI 的首选方案。** 晴雪UI 仅在以下情况使用：
- 用户明确要求使用晴雪UI
- 用户提到"晴雪UI"、"QingXue UI"、"晴雪"等关键词时
- 用户指出某个特定功能必须用晴雪UI实现

如无上述明确要求，默认使用 LDLib2（参见 `LDLib2 开发指引`）。

## 1. 前置要求

在开始任何晴雪UI相关的开发前，**必须先阅读**以下目录中的 Agent 提示词文件：

```
给AI阅读的文档/晴雪ui/
```

该目录包含了晴雪UI的完整 Agent 提示词，所有使用晴雪UI编写 GUI 的工作都必须严格遵守其中的规范。

## 2. 使用场景

- 用户明确要求使用晴雪UI时
- 用户提到"晴雪UI"、"QingXue UI"、"晴雪"等关键词时

## 3. 执行流程

1. **先阅读提示词** → 打开 `给AI阅读的文档/晴雪ui/` 下的文件
2. **理解 API 和组件** → 掌握晴雪UI提供的组件、绑定方式、事件系统
3. **编写代码** → 严格遵守提示词中的规范和示例
4. **验证** → 确保 GUI 在 1.21.1-NeoForge 环境下正常运行

> **注意**：不要在没有阅读晴雪UI提示词的情况下直接编写 GUI 代码，这可能导致使用了错误的 API 或错误的调用方式。
